# 📦 Inventory Management System (Java Swing)

## 📌 Overview
This project is a simple Inventory Management System developed using Java Swing. It allows users to manage products through a graphical user interface. Users can add, view, search, update, and delete products easily.

The project demonstrates Object-Oriented Programming (OOP), Java GUI development, and ArrayList usage.

---

## ✨ Features
- Add new products
- View all products
- Search product by ID
- Update product quantity
- Delete product by ID
- Low stock warning (quantity < 5)
- Simple and user-friendly GUI

---

## 🛠️ Technologies Used
- Java
- Java Swing
- ArrayList
- OOP Concepts

---

## ▶️ How to Run

1. Clone the repository:
   git clone <your-repo-link>

2. Open the project in any Java IDE (IntelliJ, Eclipse, NetBeans, VS Code).

3. Run:
   inventory_gui.java

4. The GUI window will open.
